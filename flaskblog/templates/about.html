Blog Utilization Tutorial


Sign up
Sign up for an account on the blog by providing your personal information and a valid email address. This will allow you to create new posts, update existing posts and delete them as well.
Create posts
Once you have an account, you can create new posts by navigating to the appropriate section and filling out the necessary fields such as the title, content, and any relevant media.
Edit/Delete
As a registered user, you have the ability to edit or delete your previously created posts by navigating to the post and selecting the appropriate action.
View Posts
Additionally, you can view other users' posts by browsing through the different categories or searching for specific content or authors.

Brief description of the technologies used


Flask
Flask is a lightweight Python web framework that provides useful tools and features for creating web applications.
Flask-SQLAlchemy
Flask-SQLAlchemy is an extension for Flask that adds support for the SQLAlchemy ORM (Object Relational Mapper) to handle database operations.
Flask-Login
Flask-Login is an extension for Flask that provides user session management for logging in and out of the application.
Flask-Bcrypt
Flask-Bcrypt is an extension for Flask that adds support for securely hashing and checking user passwords using the Bcrypt algorithm.
Flask-WTF
Flask-WTF is an extension for Flask that integrates the WTForms library for handling forms in the application.
Jinja2
Jinja2 is a template engine for Python that is commonly used with Flask to render templates and generate HTML pages.
Bootstrap
Bootstrap is a popular HTML, CSS, and JavaScript framework for developing responsive, mobile-first web applications.
Pillow
Pillow is a library for handling images in Python, including opening, manipulating and saving various image file formats.
